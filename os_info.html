Windows 10 - операционная система для персональных компьютеров и рабочих станций, разработанная корпорацией Microsoft в рамках семейства Windows NT.

    Плюсы:

  • Совместимость с большинством аппаратных компонентов на рынке
  • Высокая распространенность
  • Простота в эксплуатации
  • Большое количество программ для этой операционной системы
  • Различные формы и типы украшений

    Минусы:

  • Низкая безопасность
  • Занимает много места
  • Небольшой набор предустановленных программ в операционной системе
  • Низкая оптимизация операционной системы
  • Необходимость очистки операционной системы от "мусора"
  • Частые и бесполезные обновления
Mac OS - семейство операционных систем, производимое компанией Apple для линейки персональных компьютеров Macintosh.

    Плюсы:

  • Более простой пользовательский интерфейс
  • Отсутствие проблем с драйверами устройств
  • Безопасность
  • Меньше вредоносного ПО и уязвимостей
  • Экосистема с плотно интегрированными компонентами

    Минусы:

  • Ограниченное количество доступного софта
  • Устройства на macOS стоят дорого, но поставляются далеко не с самым мощным железом
  • Небольшой набор предустановленных программ в операционной системе
  • Нет возможности обновить железо.
Linux — семейство Unix-подобных операционных систем, использующих ядро Linux

    Плюсы:

  • Открытый исходный код
  • Безопасность
  • Нетребовательность к ресурсам
  • Не требуются драйверы оборудования
  • Бесплатная

    Минусы:

  • Сложность освоения
  • Отсутствие версий популярных программ
  • Низкая популярность
  • Отсутствие поддержки некоторого оборудования
Итог
Выбрана Windows 10. По сравнению с Linux, она более распространена, что позволяет легче найти программное обеспечение и мануалы для данной операционной системы. Также на Windows реализованы драйверы для всех устройств, в отличие от Linux. Windows обгоняет MacOS по доступности для обывателей, ввиду меньшей цены на ПК с Windows. На 60% компьютеров по всему миру установлена Windows 10.
